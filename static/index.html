Vue sur les contrats

Le site Vue sur les contrats est un outil de visualisation qui permet de consulter les contrats et les subventions octroyés par la Ville de Montréal.

Il donne accès à tous les contrats autorisés par le comité exécutif et par le conseil municipal depuis janvier 2012.
